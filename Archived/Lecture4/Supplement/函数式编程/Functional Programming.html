1. 高阶函数 (Higher-order Function)

高阶函数：若函数的参数含有其他函数，则该函数被称为高阶函数。即高阶函数能够接受其他函数作为参数传入，是一种高度抽象的编程范式。

        
        123456789
        >>> f = abs(-10)
>>> f
10
>>> g = abs
>>> g
<built-in function abs>
>>>
>>> g(-10)
10

上图中，abs(-10)是函数调用，返回函数调用的结果；而abs是函数名，指向函数本身；可以把函数名赋给其他变量，也就是把函数本身赋值给了其他变量，这样就可以通过该“其他变量”来调用这个函数，上图用g调用了abs函数。

其实也可以把函数名看成变量，如abs也是变量，如果：

        
        1234567
        >>> abs(-10)
10
>>> abs = "abs"
>>> abs(-10)
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: 'str' object is not callable

可见内置函数abs用不了了，因为这个时候abs不再指向绝对值函数了，而是指向-10，需要重启python。

具体的高阶函数

        
        12
        def add(x, y, f):
    return f(x) + f(y)

调用add(-5, 6, abs)返回11。

1) map

map函数有两个参数：

  • 函数；
  • 可迭代对象，即Iterable；

map函数将传入的函数依次作用于可迭代对象的每个元素，最终返回的结果是一个迭代器，即Iterator。

例如：

        
        1234567
        def f(x):
	return x*x

aRange = range(10)
result = map(f, aRange)
print(list(result))
# 打印出[0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

2) reduce

同样有两个参数：

  • 函数，但必须接收两个参数；
  • 可迭代对象，即Iterable；

reduce函数让前两个对象被函数处理，其结果在于后一个被函数处理，以此类推，效果为：

        
        123
        reduce(f, [x1, x2, x3, x4])
# 等价于
f(f(f(x1, x2), x3), x4)

用reduce进行序列求和：

        
        123456
        from functools import reduce
def add(x, y):
	return x+y

print(reduce(add, [1, 3, 5, 7, 9]))
# 打印出25

3) filter

用于过滤序列，两个参数：

  • 函数；
  • 可迭代对象；

filter将函数作用于可迭代对象的每个元素上，若为True则保留，若为False则舍弃，最后返回一个迭代器。

用filter求素数——埃氏筛法

  • step 1: 列出从2开始的所有自然数；
  • step 2: 取序列的第一个数，即2，它一定是素数；
  • step 3: 然后用2把序列的2的倍数筛掉；
  • step 4: 取新序列的第一个数，即3，它一定是素数；
  • step 5: 然后用3把序列的3的倍数筛掉；
  • step 6: 循环上述步骤。
        
        12345678910111213141516171819
        # 生成器，生成从3开始的奇数序列
def _odd_iter():
	n = 1
	while True:
		n += 2
		yield n

# 筛选函数，不能被n整除的返回True
def _not_divisible(n):
	return lambda x : x%n > 0

# 素数生成器
def primes():
	yield 2
	it = _odd_iter()
	while True:
		n = next(it)
		yield n
		it = filter(_not_divisible(n), it)

4) sorted

sorted函数可以直接对可迭代对象进行排序，也可以接收一个key函数来自定义排序规则，如：

        
        12
        # 返回['1', '11', '111', '1111', '11111']
sorted(['1','1111','11','111','11111'], key = len)

key参数传入的函数将作用于可迭代对象的每一个元素上，并根据返回值进行升序排序。

2. 返回函数 (Return a Function)

可以在函数内部定义函数，再将函数作为返回值返回，例如：

        
        1234567
        def lazy_sum(*args):
    def sum():
        ax = 0
        for n in args:
            ax = ax + n
        return ax
    return sum

调用lazy_sum函数后，返回的是一个函数，这个函数的参数已经设置好了，也就是“记住了”，调用这个函数的时候，会返回具体的求和值。

闭包（Closure）：内部函数sum可以引用外部函数lazy_sum的参数和局部变量，当lazy_sum返回函数sum时，相关参数和变量都保存在返回的函数中。

注意：返回的函数并没有立刻执行，直到返回的函数()调用这样才会执行。

        
        12345678910111213
        def count():
    fs = []
    for i in range(1, 4):
        def f():
             return i*i
        fs.append(f)
    return fs

f1, f2, f3 = count()

print(f1())    # 9
print(f2())    # 9
print(f3())    # 9

原因是，内部函数返回值里有循环变量，而内部函数并非立刻执行，是等待状态，所以等到3个函数都返回时，i已经是3了。

返回函数不要引用任何循环变量，或者后续会发生变化的变量

3. 匿名函数(Anonymous Function)

Python中的匿名函数就是lambda函数。

lambda x:x*x

冒号前的x表示函数的参数，冒号后的x*x表示返回值。

4. 装饰器(Decorator)

函数对象有一个__name__属性，返回函数的名字。

装饰器：不改变函数的代码，但在代码运行期间动态增加功能的方式，是装饰器。

例如：

        
        12
        def now():
	print('2015-3-25')

希望增加now函数的功能，比如在函数调用前打印日志，但又不希望改变now函数的代码，此时需要使用装饰器。

本质上，一个Decorator是一个返回函数的高阶函数。

定义log装饰器：

        
        123456
        def log(func):
	# 装饰func函数
	def wrapper(*args, **kw):
		print("call %s(): " % func.__name__)
		return func(*args, **kw)
	return wrapper

注意上面wrapper里return的不是函数对象，而是原函数对象的执行结果，即不是return func而是return func(*args, **kw)。

给函数now加上装饰器：

        
        123
        @log
def now():
	print("2015-3-25")

@log相当于now = log(now)，将now替换成了装饰过的now。

如果decorator本身需要传入参数，那就需要写一个返回decorator的高阶函数

        
        1234567
        def log(text):
    def decorator(func):
        def wrapper(*args, **kw):
            print('%s %s():' % (text, func.__name__))
            return func(*args, **kw)
        return wrapper
    return decorator

加装修饰器：

        
        123
        @log('execute')
def now():
    print('2015-3-25')

相当于：

        
        1
        now = log('execute')(now)
